NAME

xkblayout — display current xkb keyboard layout

SYNOPSIS

xkblayout [-d display] [-l]

DESCRIPTION

xkblayout is a utility that prints the current xkb keyboard layout.

The options for xkblayout are as follows:

-d display
Specify the display to use.
-l
Long format. Prints full xkb keyboard layout string.

ENVIRONMENT

DISPLAY
xkblayout use this display unless the -d option is given.

HISTORY

The xkblayout utility was developed on OpenBSD 4.4.
